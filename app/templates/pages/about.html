About

Documentation, ethical considerations, and the story behind the tool

What Is Thresh?

Thresh is a local-first research tool that wraps the Reddit API behind a guided interface. It helps graduate researchers collect, explore, and export Reddit data for dissertation work — without writing code.

Named for the biblical threshing floor: the place where grain is separated from chaff through deliberate labor.

Ethical Use

Reddit data involves real people. Before collecting, consider re-identification risks, your IRB requirements, and Reddit's API Terms of Service. Default exports anonymize usernames.
